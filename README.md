## 🛠️ متطلبات التثبيت (Installation)

لتمكين تشغيل اللعبة على جهازك، يجب تثبيت محرك Dart أولاً بناءً على نظام تشغيلك:

### **1. مستخدمو Windows:**
* **التحميل:** توجه إلى [موقع Dart الرسمي](https://dart.dev/get-dart) وحمل ملف التثبيت (SDK Installer).
* **الإعداد:** اتبع خطوات معالج التثبيت التقليدية.
* **التحقق:** افتح **PowerShell** واكتب الأمر التالي للتأكد:
  ```powershell
  dart --version
2. مستخدمو Linux (Arch / Archcraft):
يمكنك التثبيت مباشرة عبر التيرمينال باستخدام مدير الحزم pacman:

Bash
sudo pacman -S dart
📥 تحميل المشروع (Download)
يمكنك الحصول على نسخة من الكود المصدري عبر أداة Git. افتح التيرمينال (أو CMD في ويندوز) ونفذ:

Bash
# استنساخ المستودع من GitHub
git clone [https://github.com/Imad015/QUIZAPP.git](https://github.com/Imad015/QUIZAPP.git)
🚀 كيفية الفتح والتشغيل (Setup & Launch)
بعد التحميل، اتبع هذه الخطوات بالترتيب لفتح اللعبة وتشغيلها بنجاح:

الخطوة الأولى: الدخول إلى مجلد المشروع
يجب أن ينتقل التيرمينال إلى المجلد الذي يحتوي على ملفات الكود التي تم تحميلها:

Bash
cd QUIZAPP
الخطوة الثانية: تشغيل الملف الأساسي
استخدم أمر Dart لتشغيل محرك اللعبة مباشرة:

Bash
dart QuizApp.dart
🎮 نظام اللعبة (Game Logic)
يعتمد التطبيق على نظام مستويات ديناميكي:

المستوى السهل (Easy Level): يحتوي على أسئلة أساسية لتعريف المستخدم بنظام اللعبة وجمع النقاط الأولية.

المستوى المتوسط (Medium Level): يتم تفعيل هذا المستوى تلقائياً عند تحقيق المستخدم لدرجة نجاح معينة، مما يضيف طابعاً تنافسياً وتحدياً أكبر.

التحقق من الإجابات: يستخدم التطبيق منطقاً شرطياً دقيقاً لمقارنة مدخلات المستخدم بالبيانات المخزنة داخل الكلاسات البرمجية.

📁 بنية المشروع (Architecture)
تم تقسيم المشروع إلى ملفات منفصلة لاتباع منهجية Clean Code:

QuizApp.dart: يمثل قلب المشروع (The Controller). يحتوي على حلقات التكرار (Loops) الأساسية، واستقبال مدخلات المستخدم ومعالجتها.

Quesn.dart: يمثل طبقة البيانات (The Models). يحتوي على تعريف الكلاسات (Eesy, Med) والـ Constructors التي تضمن بناء الأسئلة بشكل صحيح.
